# 🧠 ManageCoreClipBoard

A simple but powerful clipboard manager built in Python.

## 🚀 Features

- Automatic clipboard tracking
- History list
- Click to restore previous clipboard entries
- Lightweight GUI (Tkinter)
- Easy to extend (storage, encryption, sync)

## 🛠 Installation

```bash
pip install -r requirements.txt

▶️ Run
Two Click on ManageCoreClipBoard.exe

📦 Project Structure
main.py → entry point
ui.py → GUI logic
clipboard.py → clipboard access
storage.py → history management
